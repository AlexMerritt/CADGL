rotating


To the right is a list of mods available on the active models. They will be loaded when a new shape is created or loaded

To the left is the UI for managing the models.


Load Model
A list is populated with all current models available to work on that are stored in the database. Click a model name to load it in the scene.

Create Model
Create a new model and saves it to the database. Click and fillout a name and new model will be created.

Save Model
Click to save and pushes all of the changes on the active model to the database.

Upload Model
Click to open a window that you can select a .obj model to load

To rotate the model right click and drag the mouse around.

Carve Mod
Cave a path into the model based on the mouse position. Click and drag around the model to add deformations to it.

Start: Start a new carve. The models dimentions will be locked to before the carve mod was started to give a consistant carve depth. When click the button will change to End.
End: End the carve and finialize the new shape. When clicked the button will change to Start.
Depth: How strong the carve will be applied. The higher the number the deeper the carve will be.

Build Mod
Build up material along a path into the model based on the mouse position. Click and drag around the model to add deformations to it.

Start: Start a new build. The models dimentions will be locked to before the carve mod was started to give a consistant carve depth. When click the button will change to End.
End: End the build and finialize the new shape. When clicked the button will change to Start.
Depth: How strong the build will be applied. The higher the number the more material that will be applied when building.
