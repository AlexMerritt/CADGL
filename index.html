To the right is a list of mods available on the active models. They will be loaded when a new shape is created or loaded

Widen
Widen the model by a constant value the selected level. Click and drag in the Level box or click the number and type in a value to set the select level.

Extrude
Extrude the model by a constant value at the select raduis. Click and drag in the Radius box or click the number and type in a value to set the select radius.

To the left is the UI for managing the models.


Load Model
A list is populated with all current models available to work on that are stored in the database. Click a model name to load it in the scene.

Create Model
Create a new model and saves it to the database. Click and fillout a name and new model will be created.

Save Model
Click to save and pushes all of the changes on the active model to the database.

To rotate the model right click and drag the mouse around.

Carve
Click the button to toggle the carving tool. When the carving tool is active click and drag the mouse and it will carve on the model to a fixed depth. Model rotation will be disabled while carving. You can tell which mode you are in based on the text directly under the render window.
